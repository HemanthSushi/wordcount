## üöÄ Features

* **Word Count:** Instantly calculates the total number of words in your text.
* **Character Count:** Calculates the total characters (including spaces).
* **Frequency Analysis:** precise breakdown of every unique word used and how many times it appears.
* **Case Insensitive:** Smart logic treats "The" and "the" as the same word.
* **Clean UI:** Built with Bootstrap 5 for a responsive, modern look.
* **Scrollable Results:** Handles large texts gracefully with a scrollable frequency list.
